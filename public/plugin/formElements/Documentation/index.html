Integration and Usage

You'll find 2 folders containing each one a separate variant of the form, one is light, the other one dark. Once you've picked up the one you want to use, the integration is straight forward :

  • First add all assets (ie and css subfolders and the file form.css.
  • If you use a different folder structure, don't forget to update all relevant links pointing to each asset.
  • Link to the file form.css into your HTML page.
  • Once done, open form.html and select the whole form to paste it into your HTML page.

If you need to edit some parts of the CSS, open form.css and you'll find all the content divided into the following sections :

00 PAGE AND GALLERY STYLING
01 GENERAL FORM STYLING
02 INPUT FIELDS & TEXT AREA
03 CHECKBOXES & RADIO BUTTONS
04 SELECT ELEMENT
05 TOGGLE BUTTONS
06 BUTTONS
07 FEEDBACK MESSAGES
08 TOOLTIPS

The first part labelled 00 PAGE AND GALLERY STYLING should be removed, it just styles the page and doesn't have any impact on the form itself.

Internet Explorer

Internet Explorer (versions 6, 7 and 8) require a few additional steps : if you followed the previous steps, you should have already included the ie subfolder into your project. Once done, you have to add the following markup in your HTML like in all the files included in the package :

<!--[if lt IE 9]>
<link rel="stylesheet" href="ie/form_ie.css">
<script src="http://ie7-js.googlecode.com/svn/version/2.1(beta4)/IE9.js"></script>
<![endif]-->

One line links to the file form_ie.css (styles applied only to IE6-8) and the other one to the IE7-js library (used to upgrade IE6-8 to be compatible with modern browsers). It's not a 100% working solution but it helps a lot displaying the form properly.

The ie subfolder contains another file named PIE.htc. It enables some CSS3 effects in IE6-8 like rounded corners or shadows. This file is linked from form_ie.css (at the end) so if you use another folder structure, make sure that you update the path to PIE.htc in form_ie.css. This path is relative to the HTML file.

That's all for IE, for more informations about PIE, see the "Sources and Credits" section.
