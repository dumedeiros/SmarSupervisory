A) Description - top

Explr is a jQuery plugin that will create a simple tree from a normal menu list with ul-li-a pattern, such as

	<ul id="tree">
	    <li>
		<a href="#">1st item 1st level</a>
		<ul>
		    <li>
			<a href="#">1st item 2nd level</a>
		    </li>
		</ul>
	    </li>
	    <li>
		<a href="#">2nd item 1st level</a>
	    </li>
	</ul>

into a hierarchical tree by simply call explr() function in javascript section:

	$('#tree').explrTree();
  • 1st item 1st level
    • 1st item 2nd level
  • 2nd item 1st level

B) Features - top

  • Simple and lightweight (4KB minified JS+CSS)
  • Each branch could be set to start as expanded/collapsed
  • 50+ custom icons ready for individual/multiple menu item(s)
  • Supports AJAX contents to be loaded from each menu item's href destination
  • Supports multiple tree in one page
  • Requires only semantic markups: ul, li, a
  • Styling is customizable via CSS
  • Cross-browser compatible
  • Session cookies (to keep the tree hierarcy state when load new page)

4. Load AJAX Content

With AJAX feature, when we click a link within the tree, the browser wont need to refresh to load a new page we requested, instead it will load asynchronously and placing the content inside an existent one of selected element. For example, we want to load all HTML content of a clicked link within tree to a div with id="ajax-content-wrapper" beside the tree, then we must set the ajaxContainerId value to 'ajax-content-wrapper'. 

	$('#id').explr({
		ajaxContainerId	: 'ajax-content-wrapper',
		treeHeight	: 200,
		treeWidth	: 400
	});

Then don't forget to add rel="explr-ajax" to every anchor link elements you wanted to be AJAXified. 

	<div class="tree-wrapper">
	    <ul id="id">
		<li>
		    <a href="#">External content loaded with AJAX</a>
		    <ul>
			<li>
			    <a href="test.html" rel="explr-ajax">load test.html</a>
			</li>
			<li>
			    <a href="test.php?getdata=hello" rel="explr-ajax">load test.php along with a GET data</a>
			</li>
		    </ul>
		</li>
		<li>
		    <a href="#">Internal content</a>
		    <ul>
			<li>
			    <a href="#toc">go to top of page</a>
			</li>
		    </ul>
		</li>
	    </ul>
	</div>
	<div id="ajax-content-wrapper"></div>

5. A Note on Session Cookies

Please test cookie feature from a webserver or localhost, as if its accessed from local folder could result fail. Also, thats the reason why this help file is not featuring this feature (rememberState option is set to false) as I think this will be accessed from your local folder.

6. All Available Options

So then, from examples above we can sum up all available options to be passed to an explrTree() function:

Option Default value Description
ajaxContainerId 'explr-content' Set this value to the id of an element which you want to be as the container of every contents of links that will be loaded through AJAX
ajaxLoadingText '<p>loading data..</p>' Set alternative HTML to be displayed when loading AJAX request
ajaxOptions {} Set your custom $.ajax() function to overriden default function when a link with rel="explr-ajax" clicked. For any arguments available see $.ajax() API reference
animDuration 'fast' Set the duration of toggle animation. To disable animation effect when collapsing/expanding tree, set this value to 0
folderTooltip 'click to expand/collapse' Set the tooltip to be displayed when mouse is hover over the folders
rememberState true Set to false if you don't want the tree to remember it's state when load new page that contain same tree
startCollapsed true Set to false only if you want the tree to be expanded at start
treeHeight 'auto' Set the height property of tree in number (of pixels) or 'auto' to adjust automatically
treeWidth 'auto' Set the width property of tree in number (of pixels) or 'auto' to adjust automatically

For example: 

	$('#id').explr({
		ajaxContainerId	: 'ajax-content-wrapper-2',
		ajaxLoadingText	: '<img src="preloader.gif" alt="please wait" />'	
		ajaxOptions	: {
			success	: function(data){
				alert(data);
			}
		},
		animDuration    : 0,    // disable animation effect
		folderTooltip	: 'tekan untuk membuka/menutup',
		rememberState	: false,
		startCollapsed	: false,
		treeHeight	: 200,
		treeWidth	: 400
	});
